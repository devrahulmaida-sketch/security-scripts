## 📁 Scripts Overview

| Script | Description | Difficulty |
|--------|-------------|------------|
| `network_scanner.py` | Scan local network for active devices | ⭐ Beginner |
| `port_scanner.py` | Scan ports on a target IP | ⭐ Beginner |
| `hash_checker.py` | Calculate & verify file hashes | ⭐ Beginner |
| `password_strength.py` | Check password strength | ⭐ Beginner |
| `subdomain_finder.py` | Find subdomains of a domain | ⭐⭐ Intermediate |
| `headers_checker.py` | Check website security headers | ⭐ Beginner |
